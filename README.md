## Stack

This project was bootstrapped with [create.t3.gg](create.t3.gg). It includes the following technologies:

- [React.js](react.dev)
- [TailwindCSS](tailwindcss.com)
- [Next.js](nextjs.org)
- [trpc](trpc.io)
- [MongoDB](mongodb.com)
- [Prisma](prisma.io)
- [TypeScript](typescriptlang.org)
